Control de acceso versión 19/03/21
========================================================

NOTAS GENERALES
---------------

Aplicación web en Java que use sesiones.

· Iniciar la aplicación con un formulario.

· Si el usuario no está dado de alta, se redigirá automáticamente al formulario de registro.

· El formulario de registro tendrá validación de la contraseña.

· Cuando el usuario se logea o se registra correctamente se dirige a la home, donde estará su nombre.

· Los usuarios pueden cerrar sesión y volver al formulario de login.

· El admin puede modificar y eliminar usuarios.

· Utilizar includes para el header y el footer.

NOTAS ESPECIFICAS
-----------------

Los usuarios que no son administradores no pueden modificar sus datos.

OTRAS CONSIDERACIONES
---------------------

No se consideran.
